## Installation
### CocoaPods:
In your `Podfile` add:

1. Add `pod 'iOSWrappers'` to your *Podfile*.
2. Install the pod(s) by running `pod install`.
3. Include iOSWrappers wherever you need it with `import iOSWrappers`.
